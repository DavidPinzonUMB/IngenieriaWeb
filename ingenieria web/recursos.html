Fuentes Utilizadas y Recomendadas

W3C - World Wide Web Consortium

Sitio oficial que define los estándares de la web. Aquí se encuentran las especificaciones de HTML, CSS y las normas de accesibilidad.

Enlace: https://www.w3.org/

W3C - Web Accessibility Initiative (WAI)

Explica las pautas de accesibilidad web (WCAG). Se utilizó para comprender la importancia del diseño accesible.

Enlace: https://www.w3.org/WAI/

MDN Web Docs

Documentación técnica sobre HTML5, CSS y JavaScript. Fue utilizada para reforzar conceptos sobre estructura semántica.

Enlace: https://developer.mozilla.org/es/

IBM - ¿Qué es cliente-servidor?

Explicación clara y general sobre cómo funciona el modelo cliente-servidor en redes y aplicaciones web. Fue consultado para explicar la arquitectura básica.

Enlace: https://www.ibm.com/topics/client-server

Validador Oficial de HTML - W3C

Herramienta utilizada para verificar que el código cumpla con los estándares oficiales de HTML5.

Enlace: https://validator.w3.org/

Wikipedia - Ingeniería Web

Fuente general para obtener una definición básica sobre el concepto de ingeniería web.

Enlace: https://es.wikipedia.org/wiki/Ingeniería_web
